#  Li-Fi Enabled Secure Communication for Submarines

A Li-Fi based underwater communication prototype that demonstrates secure wireless data transmission using **LED light** and an **LDR (Light Dependent Resistor)** instead of conventional radio-frequency communication. The system showcases the concept of optical communication for secure and interference-free data transfer.

---

##  Project Overview

This project demonstrates the concept of **Li-Fi (Light Fidelity)** communication using visible light. The transmitter converts data into light pulses through an LED, while the receiver uses an LDR sensor to detect these pulses and reconstruct the transmitted message. The prototype is inspired by underwater communication scenarios where radio-frequency signals are ineffective.

---

##  Features

-  Li-Fi based wireless communication
-  Secure optical data transmission
-  RF-free communication
-  Real-time message transfer
-  LCD display for received data
-  Arduino-based implementation
-  Low-cost hardware prototype

---

##  Components Used

- Arduino Uno (2x)
- LED
- LDR (Light Dependent Resistor)
- 16×2 LCD Display
- 220Ω Resistor
- 10kΩ Resistor
- Breadboard
- Jumper Wires
- USB Cable

---

##  Working Principle

1. The transmitter Arduino receives the input message.
2. The message is converted into LED light pulses.
3. The LED transmits the encoded optical signal.
4. The LDR detects changes in light intensity.
5. The receiver Arduino decodes the received signal.
6. The decoded message is displayed on the LCD.
